What are Transforms?

Transform is a new addition in CSS to alter and position element. It can be done in 2D or 3D. Using transforms we can Scale, Skew, Translate, and Rotate an Element.

Scale

Elements can be scaled using scale property of CSS transform.
This property works in three ways: scaleX( ), scaleY( ) and scale( ) to scale element along X-axia, Y-axis, and equally respectively

Anything greater than 1 Upscales the element, lesser than 1 downscales it while value 1 maintains original size.

1. transform:scale(2) :Doubles the size
2. transform:scaleX(2) :Doubles the size along X-axis
3. transform:scaleY(2) :Doubles the size along Y-axis

Translate

Elements can be translated/moved using translate property of CSS transform.
This property works in three ways: translateX( ), translateY( ) and translate(x,y ) to scale element along X-axia, Y-axis, and along x and y axes respectively
If the second parameter is not specified, it is taken 0 by default.

1. transform:translate(200px, 300px) :Moves the element 200px along X-axis and 300px along Y-axis
2. transform:translateX(200px) :Moves the element along positive X-axis
3. transform:translateY(200px) :Moves the element along positive Y-axis

Skew

Elements can be skewed using skew property of CSS transform.
This property works in three ways: skewX( ), skewY( ) and skew(x,y) to skew element along X-axis, Y-axis, and equally respectively

If the second parameter is not specified, it is taken 0 by default.

1. transform:skew(20px) : Skews the element 20px along X-axis
2. transform:skewX(20px) : Skews the element 20px along X-axis
3. transform:skewY(20px) : Skews the element 20px along Y-axis

Rotate

Elements can be rotated using rotate property of CSS transform.
This property works in two ways: rotate(-10deg ) and rotate(10deg ) to rotate counter-clockwise and clockwise respectively.

1. transform:rotate(-30deg) :Rotates the element counter-clockwise
2. transform:rotate(30deg) :Rotates the element clockwise
